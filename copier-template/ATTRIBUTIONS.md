# Attribution/Credit

In this file, you can attribute and give credits.

## Example

```markdown
This app includes:

* components from [xxx](https://example.com/xxx/xxx) used under [MIT license](https://example.com/xxx/xxx/LICENSE.md).
* photos from [Unsplash](https://unsplash.com) used under [license](https://unsplash.com/license).
```
